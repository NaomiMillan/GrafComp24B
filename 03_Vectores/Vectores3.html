Quiver¶

La funcin plt.quiver en Matplotlib se utiliza para crear grficos de campos de vectores bidimensionales.

Sintaxis¶

plt.quiver(X, Y, U, V)

X, Y: Coordenadas de los puntos en los que se dibujarn los vectores.

U, V: Componentes del vector (en las direcciones X e Y) en cada punto.

axvline() y axhline()¶

Las funciones plt.axvline() y plt.axhline() en Matplotlib se utilizan para dibujar lneas verticales y horizontales en un grfico.

xlim() y ylim()¶

Las funciones plt.xlim() y plt.ylim() en Matplotlib se utilizan para establecer los lmites de los ejes X e Y en un grfico.

In [3]:
plt.quiver([0,0],[0,0],[1,2],[4,2], scale_units='xy', angles='xy', scale=1)
plt.xlim(0,4)
plt.ylim(0,4)
Out[3]:
(0.0, 4.0)

En la grfica anterior se crean 2 vectores que parten del origen hacia las coordenadas (1,4) y (2,2).

El plano se limita a las medidas 4 en X y 4 en Y

def grafvecs(vecs, cols):
    plt.figure(figsize=(5,5))
    # vecs: lista de vectores
    for i in range(len(vecs)):
        vec = vecs[i]
        plt.quiver(0,0, vec[0], vec[1], color = cols[i], 
                   angles='xy', scale_units='xy', scale=1)

Explicación de la función¶

La funcin grafvecs(vecs, cols) grafica una lista de vectores en un plano bidimensional utilizando la funcin plt.quiver()

La funcin recibe dos parmetros:

 vecs: Una lista de vectores, donde cada vector es una lista de dos elementos que representan las componentes xx y yy del vector.
 cols: Una lista de colores, donde cada elemento corresponde al color que tendr cada vector en el grfico.

Se crea una figura de 5x5.

Se utiliza plt.quiver() para dibujar un vector desde el origen hasta las coordenadas dadas por las componentes del vector vec[0] (componente xx) y vec[1] (componente yy).

color=cols[i]: Asigna el color para el vector.

angles='xy': Especifica que las componentes del vector estn en el sistema de coordenadas cartesiano.

scale_units='xy': Mantiene las escalas de los ejes X y Y consistentes.

scale=1: Mantiene la escala en 1.

In [7]:
v1 = np.array([1,2])
v2 = np.array([3,4])
v3 = np.array([8,4])

grafvecs([v1, v2, v3], ['blue', 'red', 'black'])
plt.xlim(0, max(v1[0], v2[0], v3[0]))
plt.ylim(0, max(v1[1], v2[1], v3[1]))
Out[7]:
(0.0, 4.0)

plt.xlim(0, max(v1[0], v2[0], v3[0])): establece el lmite del eje X desde 0 hasta el valor mximo entre las primeras componentes de los vectores (1, 3 y 8), por lo que el eje X va de 0 a 8.

plt.ylim(0, max(v1[1], v2[1], v3[1])): Establece el lmite del eje Y desde 0 hasta el valor mximo entre las segundas componentes de los vectores (2, 4, y 4), por lo que el eje Y va de 0 a 4.

La grfica muestra tres vectores que parten del origen (0,0)

  • El vector azul va hacia la coordenada (1,2).
  • El vector rojo se dirige hacia (3,4).
  • El vector negro se extiende hasta (8,4).
