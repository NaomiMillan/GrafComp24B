¿Qué es el modelo?¶

Es un modelo que representa la interaccin entre dos especies, presa y depredador, en un ambiente controlado. Para buscar un equilibrio entre ambas especies.

Estado original¶

En este ejemplo se inicia con una poblacin de depredadores mayor (0.5) a comparacin de la poblacin inicial de presas (0.1), sin embargo, el factor de aumento natural de las presas es mucho mayor, es decir, que se reproducen ms rpido. dt = 0.001; max_time = 100

# initial time and populations
t = 0; x = 1.0; y = 0.5

# Resto del cdigo donde se definen las ecuaciones de Lotka - Volterra y se muestra la grfica
In [8]:
# empty lists in which to store time and populations
t_list = []; x_list = []; y_list = []

# initialize lists
t_list.append(t); x_list.append(x); y_list.append(y)

while t < max_time:
    # calc new values for t, x, y
    t = t + dt
    x = x + (a*x - b*x*y)*dt
    y = y + (-c*y + e*x*y)*dt

    # store new values in lists
    t_list.append(t)
    x_list.append(x)
    y_list.append(y)

# Plot the results    
p = plt.plot(t_list, x_list, 'r', t_list, y_list, 'g', linewidth = 2)

Estado 4¶

Ahora bien, si queremos que sean los depredadores quienes estn en el pico mximo de poblacin, podemos hacer lo siguiente. Comenzamos con una poblacin inicial igual para ambas especies, y definimos su ritmo de crecimiento con el mismo ndice, para mantener a las especies en niveles similares, sin embargo, para lograr que los depredadores sean la especie dominante debemos aumentar el parmetro de muerte de las presas por depredadores, en este caso en 0.7. Estado 5¶

En este ejemplo, se inicializaron ambas poblaciones (presas y depredadores) en 0.5 y se ajustaron todos los parmetros para que tuvieran el mismo valor. Esto asegura que tanto las presas como los depredadores se reproducen al mismo ritmo. Por lo que si solo considerramos estas condiciones el crecimiento de ambas poblaciones sera idntico. Sin embargo, al tomar en cuenta las interacciones entre las dos especies, esta igualdad se altera. Para mantener en equilibrio las condiciones, se ha puesto el mismo valor (0.3) para los factores de muerte para ambas especies. Esto implica que los valores de aparicin y desaparicin de presas y depredadores son iguales, pero no simultneo. La ausencia de presas provoca la disminucin de los depredadores, mientras que la falta de depredadores permite que las proliferacin de las presas, lo que a su vez, el lleva a un incremento de depredadores. Entonces es de suponer que la grfica que se obtiene muestra patrones similares de aumento y disminucin para presas y depredadores pero de forma desfasada.
